# 手动验收清单

验收设备要求：至少两部不同品牌安卓手机，三个微信账号分别扮演创建者（Owner）、管理员（Admin）和查看成员（Viewer）。

---

## 1. 家庭创建与邀请

- [ ] **手机 A（Owner）**：创建家庭，填写孩子昵称，进入首页
- [ ] **手机 A**：生成邀请链接，确认有效期为 24 小时
- [ ] **手机 B（Admin）**：通过邀请链接加入家庭，角色为 viewer
- [ ] **手机 A**：将 Admin 提升为管理员
- [ ] **手机 C（Viewer）**：通过新邀请链接加入，角色为 viewer
- [ ] **手机 A**：确认成员列表显示全部三位成员

## 2. 权限验证

- [ ] **手机 C（Viewer）**：确认无法看到"添加课程"按钮
- [ ] **手机 C**：确认无法编辑课程或完成课程
- [ ] **手机 B（Admin）**：确认可以添加和编辑课程
- [ ] **手机 A**：确认可以移除成员（不能移除自己）

## 3. 课程创建与排课

- [ ] **手机 A**：添加课程"外教口语"，剩余 8 课时，每周六 10:00
- [ ] **手机 A**：选择提醒成员（仅 Owner 和 Admin）
- [ ] **手机 B**：确认在首页看到相同课程和剩余课时
- [ ] **手机 C**：确认能看到课程但不能编辑

## 4. 日历与课程实例

- [ ] **手机 A**：进入日历 tab，确认本周六有"外教口语"课程
- [ ] **手机 A**：点击课程进入调整页，确认显示原定时间
- [ ] **手机 A**：选择"改期"，改为周六 14:00，确认仅修改本次
- [ ] **手机 B**：确认日历中该课程时间已更新
- [ ] **手机 A**：再次点击该课程，选择"请假"，确认状态变为"已请假"

## 5. 完成课程与扣课

- [ ] **手机 A**：在首页点击"完成本次课程"
- [ ] **手机 A**：确认页面显示"8 → 7"余额变化
- [ ] **手机 A**：点击"仅扣除课时"
- [ ] **手机 A**：确认首页剩余课时变为 7
- [ ] **手机 B（Admin）**：确认剩余课时同步为 7

## 6. 并发与幂等

- [ ] **手机 A 和手机 B**：同时点击完成同一节课
- [ ] 确认仅扣一次课时（不重复扣减）
- [ ] 确认后操作者看到"课程已完成"提示

## 7. 撤销扣课

- [ ] **手机 A**：撤销刚才的扣课操作
- [ ] 确认剩余课时恢复为 8
- [ ] 确认操作日志中新增撤销记录

## 8. 学习记录

- [ ] **手机 A**：完成课程后选择"完成并添加学习记录"
- [ ] **手机 A**：填写主题、笔记、标签
- [ ] **手机 A**：保存记录，确认返回成长时间线
- [ ] **手机 B**：确认成长页面能看到新记录
- [ ] **手机 C（Viewer）**：确认能看到记录但无法编辑

## 9. 草稿与上传

- [ ] **手机 A**：编辑学习记录时切到后台再切回，确认草稿未丢失
- [ ] **手机 A**：上传一张图片，确认显示上传进度
- [ ] **手机 A**：上传一个 PDF，确认显示在记录中

## 10. 配额保护

- [ ] 手动将存储用量填至 80%，确认显示警告
- [ ] 手动将存储用量填至 95%，确认新文件上传被阻断
- [ ] 确认文字记录仍可正常保存

## 11. 数据导出

- [ ] **手机 A（Owner）**：在家庭页点击"导出数据"
- [ ] 确认导出内容包含课程、课表、学习记录、操作日志
- [ ] 确认 Viewer 手机无导出按钮

## 12. 成员移除与权限失效

- [ ] **手机 A**：移除 Viewer 成员
- [ ] **手机 C（原 Viewer）**：确认无法再查看家庭数据
- [ ] **手机 C**：确认无法获取新的文件访问地址

## 13. 家庭删除与恢复

- [ ] **手机 A**：进入家庭页，点击"删除家庭"
- [ ] 输入错误昵称，确认无法删除
- [ ] 输入正确昵称，确认进入 30 天延迟删除
- [ ] 确认可以撤销删除

## 14. 弱网与异常

- [ ] 弱网环境下点击"完成课程"，确认不出现重复扣课
- [ ] 断网后尝试编辑，确认有明确的错误提示
- [ ] 快速连续点击"保存"按钮，确认不产生重复记录
